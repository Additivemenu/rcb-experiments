# Simple Next.js Website with shadcn UI

A clean and modern website built with Next.js, TypeScript, and shadcn UI components.

## Features

- Built with Next.js App Router
- Fully typed with TypeScript
- Beautiful UI components from shadcn UI
- Responsive design with Tailwind CSS
- Form validation with Zod and React Hook Form

## Pages

- **Home** - Landing page showcasing the website's features
- **About** - Information about the company and team
- **Contact** - Contact form with validation

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd next-shadcn-app
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
