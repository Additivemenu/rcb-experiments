# Simple Next.js App

A clean, modern UI built with Next.js 14.2.7 (App Router), TypeScript, and shadcn/ui components.

## Technologies Used

- **Next.js 14.2.7** - Using the App Router for improved routing and layouts
- **TypeScript** - For type safety and better developer experience
- **shadcn/ui** - Beautiful, accessible UI components built with Radix UI and Tailwind CSS
- **Tailwind CSS** - Utility-first CSS framework
- **pnpm** - Fast, disk space efficient package manager

## Features

- Modern, responsive design
- Dark mode support
- Accessible UI components
- Type-safe code

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- pnpm

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/simple-nextjs-app.git
cd simple-nextjs-app
```

2. Install dependencies

```bash
pnpm install
```

3. Run the development server

```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable UI components
  - `ui/` - shadcn/ui components
- `public/` - Static assets
- `lib/` - Utility functions and shared code
